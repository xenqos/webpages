Césure

перенос слов

Слова переносятся по слогам
 em-prun-ter, ren-ta-bi-li-té
Не оставляется на строке 1-й однобуквенный слог
 a//mortissement
Не переносится конечный слог с немой гласной
 adres//se, publi//que
Не переносится две гласные (за исключением приставок)
 monsi//eur, expéri//ence
Переносится две гласные в приставках
 pré-enquête
Переносится «x», «y» с последующей согласной
 pay-san, tex-tile, asy-métrie
Не переносится «x», «y» с последующей гласной
 e//x//emplaire, mo//y//en
Переносится если «x» обозначает [z]
 di-xième, deu-xième
Переносятся две согласные в середине слова
 bal-lon, fer-mer
Не переносится «gh», «ph», «rh», «th», «ill», «ps», «gn», «ch» (oдин звук)
 pi-gnon, rat-ta-cher
Не переносится «r» и «l» после согласной (кроме сочетания rl)
 ta-ble
Переносится сочетание «rl»
 par-lement
Переносится 3-я согласная из трёх
 ins-tituteur
Переносится после 1-ой согласной, если 2-я и 3-я обозначают один звук
 am-phibie, mar-chand
Переносится после дефиса
 ci-des//sus, micro-ordi//nateur
Не переносится после апострофа
 presqu’//île, aujourd’//hui
Переносится после первого дефиса в глагольных инверсиях с «-t-»
 a-t-//il, pense-t-//elle
Не переносятся имена и фамилии после сокращенний, обращений, инициалов
 M. // Morin, M. // et// Mme // Morin
Не переносятся числительное перед и после существительного
 18 // juin // 20XX, article // VII
Не переносятся аббревиатуры, символы, матформулы, знаки
 km/h
Рекомендуется избегать переноса слов на трех строках подряд
