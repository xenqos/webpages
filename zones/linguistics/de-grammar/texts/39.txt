###### Глаголы

# Приставки частично отделяемые

%[m|durch]-
> движение через, завершение, проникновение
>> %[m|durchgehen] проходить
>> %[m|durchlesen] читать
>> %[m|durchdringen] проникать

%[m|über]-
> движение поверх, перемену места, превышение уровня
>> %[m|überlegen] класть поверх
>> %[m|überfliegen] перелетать
>> %[m|überfließen] переливаться через край

%[m|um]-
> движение вокруг, изменение, окружение
>> %[m|umgehen] ходить вокруг
>> %[m|umstellen] переставлять
>> %[m|umbauen] перестраивать

%[m|unter]-
> движение под, размещение, подчинение
>> %[m|unterschreiben] подписывать
>> %[m|unterteilen] разделять
>> %[m|unterordnen] подчинять

%[m|wider]-
> противодействие, возражение, ответное действие
>> %[m|widerstehen] устоять
>> %[m|widersprechen] возражать
>> %[m|widerhallen] резонировать, отзываться

%[m|wieder]-
> возвращение, восстановление, повторяющееся действие
>> %[m|wiederholen] повторить
>> %[m|wiedergeben] возвращать
>> %[m|wiedersehen] вновь увидеть

%[m|voll]-
> полностью, сполна
>> %[m|vollenden] завершать
>> %[m|vollziehen] исполнять
>> %[m|vollbringen] осуществлять

%[m|hinter]-
> движения назад, размещение позади, тайное действие
>> %[m|hintergehen] обманывать
>> %[m|hinterlegen] отдавать на хранение
>> %[m|hinterbringen] доносить, тайно сообщать
